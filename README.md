# FiveM Telemetry Broadcaster

This Lua script acts as a broadcaster for FXServer telemetry, sending server and player data to a custom webhook.

## Setup
1. Upload the files to your server's resource folder.
2. Update the `WebhookUrl` and `AuthToken` in `server.lua`.
3. Add `ensure fivem-telemetry-broadcaster` to your `server.cfg`.
